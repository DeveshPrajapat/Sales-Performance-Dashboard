## Overview

This project is a professional Data Analytics and Business Intelligence solution built using **Power BI** and a retail SuperStore sales dataset. The dashboard provides interactive visual insights into sales performance, customer behavior, profitability, shipping trends, and regional analysis.

The primary goal of this project is to transform raw transactional data into meaningful business insights that support strategic decision-making.

---

# Project Objectives

* Analyze overall sales and profit performance
* Identify high-performing and low-performing product categories
* Track regional and state-wise sales distribution
* Understand customer segments and purchasing behavior
* Evaluate shipping modes and payment preferences
* Detect loss-making products or regions
* Create an interactive and user-friendly dashboard for business reporting

### Dataset Features

The dataset contains retail sales transaction records including:

* Order Details
* Customer Information
* Product Categories
* Sales & Profit
* Quantity Sold
* Shipping Details
* Regional Information
* Payment Modes

### Important Columns

| Column Name        | Description                 |
| ------------------ | --------------------------- |
| Order ID           | Unique order identifier     |
| Order Date         | Date of order placement     |
| Ship Date          | Shipping date               |
| Ship Mode          | Delivery method             |
| Customer Name      | Customer details            |
| Segment            | Customer segment            |
| Country/State/City | Geographic information      |
| Category           | Product category            |
| Sub-Category       | Product sub-category        |
| Sales              | Revenue generated           |
| Quantity           | Number of items sold        |
| Profit             | Profit earned/loss incurred |
| Payment Mode       | Payment method used         |

# How to Use the Project

## Step 1: Open Dashboard

Open the `dashboard.pbix` file using Power BI Desktop.

## Step 2: Load Dataset

Ensure the dataset path is correctly connected to:

```text
SuperStore_Sales_Dataset.csv
```

## Step 3: Refresh Data

Click on:

```text
Home → Refresh
```

## Step 4: Explore Dashboard

Use slicers and filters to analyze:

* Sales trends
* Profitability
* Customer segments
* Product categories
* Regional performance
